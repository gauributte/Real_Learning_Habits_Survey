# Real Learning Habits Analysis Dashboard

## Overview
A data analytics project to understand how students actually learn 
outside the classroom. Collected 50+ responses using Google Forms 
and analyzed results using Excel and Power BI with a live data connection.

## Key Findings
- 81.13% of respondents believe lectures alone are not enough
- 37.74% learn best from YouTube over traditional lectures
- Only 16.98% actually understand what is taught in class
- 66% do their real studying right before exams
- 35.85% almost always attend class but still don't understand

## Dashboard Features
- Live KPI cards showing total responses and key percentages
- Where learning actually happens — bar chart
- Are lectures enough — pie chart with 81% saying No
- How people actually learn — bar chart of learning styles
- When real study happens — donut chart by group
- Attendance vs Understanding — cross analysis stacked bar chart
- Group slicer to filter by College, School and Working

## Tools Used
- Google Forms — survey design and data collection
- Microsoft Excel — data cleaning and COUNTIF analysis
- Power BI Desktop — interactive dashboard building
- Google Sheets — live data pipeline to Power BI
